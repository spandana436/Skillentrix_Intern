# Skillentrix_Intern
This repository contains the projects completed during my Data Science internship at Skillentrix Technologies. The projects are implemented using Python, Machine Learning, and Natural Language Processing techniques.

# Projects Included

## 1. Customer Churn Prediction

### Description
Developed a machine learning model to predict whether a customer will leave a telecom company using classification algorithms.

### Dataset Used
* TelcoCustomerChurnDataset.csv
  
### Procedure
* Data cleaning and preprocessing
* Handling missing values
* Feature engineering
* Encoding categorical data
* Feature scaling
* Model training using:
  * Logistic Regression
  * Random Forest
  * XGBoost
* Model evaluation using:
  * Accuracy
  * ROC-AUC Score
  * Precision and Recall

### Files
* Customer_churn_prediction.ipynb
* TelcoCustomerChurnDataset.csv
